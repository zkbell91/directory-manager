Directory Details Format

Your CSV should have these columns:

  • Therapist - Therapist name
  • Directory - Directory site name
  • Directory URL - Profile URL
  • Username - Login username
  • Password - Login password
  • Notes - Additional notes
Directory Grid Format

Your CSV should have:

  • Directory column with directory names
  • Therapist name columns with profile URLs
  • Empty cells for missing profiles
